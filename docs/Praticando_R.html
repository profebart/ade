Praticando o R - Módulo de Nivelamento

Introdução aos comandos básicos matemáticos:

Operações básicas:

  • Adição:+

  • Multiplicação: *

  • Divisão /

  • Potenciação: ^

  • Raiz quadrada: sqrt()

2+2 
[1] 4
2*5 
[1] 10
36/6
[1] 6
4^2 
[1] 16
sqrt(81)
[1] 9

Criando vetores numéricos

Objetos podem ser criados designando nomes aleatórios, mas que representem a ação desejada. Em geral, há duas sintaxes úteis:

  • {nome do vetor} <- c( digitar os dados)

  • {nome do vetor} = c( digitar os dados)

Tanto o sinal “<-” quanto “=” atribuem os dados entre parênteses ao vetor {Nome} criado. Atribuindo dados numéricos ao vetor “dados” e tirando a raiz quadrada de cada número do vetor, e elevando à terceira potência:

dados<-c(2, 4, 6, 6, 5, 3, 1) 
dados
[1] 2 4 6 6 5 3 1
sqrt(dados)
[1] 1.414214 2.000000 2.449490 2.449490 2.236068 1.732051 1.000000
dados3 = dados^3
dados3
[1]   8  64 216 216 125  27   1

Pode ser realizado operações entre vetores:

objeto1<- c(1, 2, 3, 4)
objeto2<- c(2, 3, 4, 5)
objeto3 <- objeto2-objeto1
objeto3
[1] 1 1 1 1

O comando “Class” identifica o tipo de dados armazenados no vetor criado:

class(dados)
[1] "numeric"
frutas<-c("laranja", "morango", "uva") 

class(frutas)
[1] "character"

Ao se emitir um comando onde não é possível realizar operação matemátiuca, surgirá o seguinte alerta:

sqrt(-1) #NaN = Not a Number!
Warning in sqrt(-1): NaNs produzidos
[1] NaN

Realizando cálculos com vetores criados

Inverso de x: criando um vetor aleatório “x”, ordenando o vetor (sort), determinando o seu tamanho (length) e determinando o seu valor inverso, arredondando casas decimais (round):

x<- c(10.4, 5.6, 3.1, 6.4, 21.7)

x_ordenado_crescente<-sort(x) 

x_ordenado_crescente
[1]  3.1  5.6  6.4 10.4 21.7
x_ordenado_decrescente<-sort(x, decreasing = TRUE) 

x_ordenado_decrescente
[1] 21.7 10.4  6.4  5.6  3.1
length(x)
[1] 5
length(frutas)
[1] 3
inverso_x<- 1/x

inverso_x
[1] 0.09615385 0.17857143 0.32258065 0.15625000 0.04608295
round(inverso_x,4)
[1] 0.0962 0.1786 0.3226 0.1562 0.0461

Média de X

Para se determinar isoladamente a média de valores de um vetor, tem-se a possibilidade de usar o comando “mean” ou calcular com a fórmula da média:

mean(x) 
[1] 9.44
form_media<- sum(x)/length(x) 
form_media
[1] 9.44
media<- sum(x)/length(x) 
media
[1] 9.44

Desvio-padrão de x

O desvio-padrão de dados de um vetor pode ser determinado pelo comando “sd”:

sd(x)
[1] 7.33846

Mediana de “dados”

A mediana de dados de um vetor pode ser determinado pelo comando “median”:

dados_ordenados<-sort(dados) 
dados_ordenados
[1] 1 2 3 4 5 6 6
median(dados) 
[1] 4
median(x) 
[1] 6.4

Variância de x

A variância de dados de um vetor pode ser determinado pelo comando “var”, ou pela fórmula estatística:

var(x) 
[1] 53.853
sd(x)^2
[1] 53.853
variancia<-(sum((x-mean(x))^2)/(length(x)-1)) 
variancia
[1] 53.853

Criando novo vetor, utilizando um anterior criado:

Pode ser mesclado vários vetores em um único novo vetor, com ou sem comandos matemáticos, inclusive adicionando números novos ao vetor (valor 0 no vetor y abaixo):

y<-c(0, 2*x) 
y
[1]  0.0 20.8 11.2  6.2 12.8 43.4
length(y)
[1] 6
z<-(c(y, 10, 100, 1000))
z
[1]    0.0   20.8   11.2    6.2   12.8   43.4   10.0  100.0 1000.0

Criando vetores de repetição:

Quando for necessário criar vetoresd com números repetidos, ou alterar uma posição de um vetor qualquer, seja por erro de digitação ou necessidade de atualizar determinado valor ou posição no vetor, temos:

VetorRep<-rep(0, 5) 
VetorRep 
[1] 0 0 0 0 0
VetorRep<-rep(1, 5) 
VetorRep 
[1] 1 1 1 1 1
VetorRep[3]<-0 
VetorRep
[1] 1 1 0 1 1

Criando Vetores lógicos

Pode ser criado vetores com comandos lógicos e com testes de Verdadeiro ou Falso para gumas posições do vetor. Assumindo que o vetor x tenha valores maiores do que 10, cria-se o vetor test_x abaixo e em seguida, testa-se as posições do vetor com dados diferentes (“!=5.6”) de 5.6 e, por último, iguais a 5.6:

x 
[1] 10.4  5.6  3.1  6.4 21.7
test_x<-x>10 
test_x 
[1]  TRUE FALSE FALSE FALSE  TRUE
test_xdif<-x!=5.6 
test_x 
[1]  TRUE FALSE FALSE FALSE  TRUE
test_x<-x==5.6 
test_x 
[1] FALSE  TRUE FALSE FALSE FALSE

Atenção:

test_x<-x=5.6 # Erro!

O sinal “=” aqui sozinho siginificaria atribuição, e não igualdade…

Descobrindo a posição onde um vetor está com formatação errônea (“NA”):

z<-c(1:3, NA) 
z
[1]  1  2  3 NA
ind<-is.na(z) 
ind
[1] FALSE FALSE FALSE  TRUE

Excluindo valores do vetor

NO eemplo a seguir, excluiu-se as posições de 1 a 3 do vetor x (sinal negativo ao lado dos parênteses no vetor), e depois, extraíu-se as posições de 1 a 3 do vetor x original:

x 
[1] 10.4  5.6  3.1  6.4 21.7
VetorRep<-x[-(1:3)] 
VetorRep 
[1]  6.4 21.7
x 
[1] 10.4  5.6  3.1  6.4 21.7
VetorRep<-x[(1:3)] 
VetorRep
[1] 10.4  5.6  3.1

Vetores alfanuméricos

Pode ser necessário criar vetores codificados, com combinações de letras e números. Para isso se utiliza o comando “paste”. O comando sep=“” ajusta a distância entre as letras e números para ficarem juntas, em intervalo de posição entre elas. Igualmente, pode-se adicionar outras pontuações entre elas, como o ponto (.) por exemplo. O comando “rep” cria repetições no padrão desejado por tantas vezes necessárias (times):

labs<-paste(c("X","Y"), 1:10) 
labs 
 [1] "X 1"  "Y 2"  "X 3"  "Y 4"  "X 5"  "Y 6"  "X 7"  "Y 8"  "X 9"  "Y 10"
labs<-paste(c("X","Y"), 1:10, sep="") 
labs 
 [1] "X1"  "Y2"  "X3"  "Y4"  "X5"  "Y6"  "X7"  "Y8"  "X9"  "Y10"
labs<-paste(c("X","Y"), 1:10, sep=".") 
labs 
 [1] "X.1"  "Y.2"  "X.3"  "Y.4"  "X.5"  "Y.6"  "X.7"  "Y.8"  "X.9"  "Y.10"
labs<-c("X", "Y")[rep(c(1,2,2,1), times=4)] 
labs
 [1] "X" "Y" "Y" "X" "X" "Y" "Y" "X" "X" "Y" "Y" "X" "X" "Y" "Y" "X"

Vetores com números e letras/palavras

O comando names associa posições de vetores entre si. Abaixo, ada número do vetor fruta foi associado a uma palavra no vetor names(fruta). Ao se criar o vertor jantar, busca-se as posições 3 e 1 respectivamente, ou seja, maçã = 1 e laranja = 5:

fruta<-c(5, 10, 1, 20) 
names(fruta)<-c("laranja", "banana", "maçã", "pêra") 
jantar<-fruta[c("maçã", "laranja")] 
jantar
   maçã laranja 
      1       5
